🌟 About the Project

This project combines Image Captioning + Image Segmentation to create a powerful AI system that can:

📝 Describe what is happening in an image
🎨 Segment and highlight objects inside the image
🤖 Use Deep Learning models to understand visual + textual data
🌐 Produce real-time outputs (optional: deploy as a web app)

It’s built for hands-on learning, research exposure, and real-world AI experience.

🎯 Key Features

✔️ Generate accurate captions from images
✔️ Perform semantic & instance segmentation
✔️ Works with standard datasets
✔️ Modular, clean & scalable code
✔️ Built using modern DL models (CNN-LSTM, Transformers, Mask R-CNN)

📂 Project Workflow
📸 Image → 🎯 Feature Extraction → 🧠 Caption Generation (LSTM/Transformer)
                                                   ↘
                                                     🎨 Segmentation (U-Net/Mask R-CNN)
